# Starter template (using webpack)

## Quick Start

#### Copy repository:
```bash
git clone https://github.com/naziks/starter-template/
```

or with ssh:
```bash
git clone git@github.com:naziks/starter-template.git
```

#### Move to local directory:
```bash
cd ./starter-template
```

#### Install dependencies:
```bash
npm install
```

#### Run:
```bash
npm run dev
```
